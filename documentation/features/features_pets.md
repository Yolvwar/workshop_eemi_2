# 🐾 APWAP - PETS MODULE - Gestion Complète des Profils Animaux

## 🐕 MODULE PETS - Profils & Suivi Personnalisé

Le module Pets est le cœur de l'application APWAP. Il crée un profil complet et évolutif pour chaque animal, servant de base à toutes les recommandations et services personnalisés basés sur les 6 piliers du bien-être animal.
